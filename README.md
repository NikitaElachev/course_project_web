# Веб-приложение для управления личными финансами

**Рабочая версия проекта доступна по ссылке:** [https://courseproject-indol.vercel.app/](https://courseproject-indol.vercel.app/)

Учебный курсовой проект по дисциплине «Разработка веб-приложений». Представляет собой легковесное клиент-серверное веб-приложение (Single Page Application) для автоматизации учета личных финансов, контроля доходов и расходов по категориям, а также формирования визуальных и экспортируемых отчетов.

## Ключевые возможности

- **Счета и транзакции:** Создание неограниченного числа счетов (наличные, кредитные карты с отрицательным балансом) и удобное добавление доходов/расходов.
- **Пользовательские категории:** Управление собственными категориями трат и поступлений с визуальным выделением (SVG-иконки).
- **Динамический расчет баланса:** Текущий остаток высчитывается на лету на основе истории операций, что полностью исключает риск рассинхронизации данных.
- **Аналитика и дашборд:** Наглядная кольцевая диаграмма расходов за текущий месяц и гибкая фильтрация операций (по счету, дате, типу и тексту).
- **Экспорт в PDF:** Мгновенная локальная генерация финансовых выписок в браузере с полной поддержкой кириллицы (без вычислительной нагрузки на сервер).
- **Ролевая модель:** Поддержка основного пользователя (`user` — полные права) и наблюдателя (`viewer` — режим "только чтение" для совместного или семейного контроля бюджета).

## Технологический стек

- **Frontend:** React, TypeScript, `framer-motion` (анимации), `lucide-react` (иконки), `jsPDF` (отчеты).
- **Backend:** Python 3, FastAPI, Pydantic, `bcrypt`.
- **База данных:** MySQL.
- **Инфраструктура (Деплой):** Vercel (фронтенд), Render (бэкенд API), Aiven (СУБД).
